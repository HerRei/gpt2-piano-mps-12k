GPT-2 Piano MPS 12k

A repo for training and sampling a GPT-2 style piano model from a 12k-scale MIDI collection, with a simple batch pipeline for comparing epoch 2 and epoch 4 generations.

Quick Start Pipeline

What This Repo Does

Data Prep

Splits raw MIDI files, augments the training set with safe piano transpositions, and tokenizes the result with Miditok REMI.

Training

Trains a 12-layer GPT-2 style model on 2048-token windows with MPS-enabled training and checkpointed epoch saves.

Generation

Samples new piano continuations from saved checkpoints using either built-in prompt profiles or a custom seed MIDI.

Quick Start

python3 scripts/prepare_12k_split.py
python3 scripts/augment_train_transpose.py
python3 scripts/tokenize_12k_augmented.py
python3 scripts/train_gpt2_piano_12k.py
python3 scripts/generation_pipeline.py --dry-run

The repository keeps large local assets out of Git. Data, checkpoints, exports, and logs remain local by design.

Human-Usable Pipeline

The batch generator is aimed at real usage rather than raw script plumbing. It can list prompt profiles, dry-run planned jobs, and run both epoch checkpoints in one command. 

python3 scripts/generation_pipeline.py --list-profiles
python3 scripts/generation_pipeline.py --dry-run
python3 scripts/generation_pipeline.py --epochs 2,4
python3 scripts/generation_pipeline.py --epochs 4 --prompt-midi /absolute/path/to/seed.mid
